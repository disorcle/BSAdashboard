# Dashboard Brasileirão Série A

Aplicação destinada à mostrar estatísticas e dados da Liga de futebol Brasileirão (série A). 

Nota: Por limitações da versão gratuita da API, o programa foi aplicado apenas nos cinco times com maior pontuação.

## Funções

- Tabela de ranking que mostra a pontuação, partidas jogadas,  vitórias, empates, derrotas, gols marcados, gols sofridos, e saldo de gols.

- Carrossel interativo, permitindo rápida alternância entre as diferentes estatísticas de cada time.

- Gráficos de pizza mostrando proporção entre vitórias/derrotas/empates e gols sofridos/gols marcados

- Gráfico em barra horizontal mostrando visualmente a disparidade de pontos entre os times.

## Tecnologias

- **React 19** + **Javascript**
- **Vite**
- **Apache ECharts** (via `echarts-for-react`)
- **Embla Carousel** (via `embla-carousel-react`)
- **Axios** (requisições à API gratuita thesportsdb)
